How to Use the Facts API

Use the API to get random facts or facts from various categories. Here's how you can get started:

curl -H "Authorization: YOUR_API_KEY" "http://localhost:8080/api/fact?type=random"

Replace YOUR_API_KEY with your generated API key.

Available Categories:

  • animals - Random animal facts
  • history - Random historical facts
  • science - Random science facts
  • space - Random space facts
  • technology - Random technology facts
  • random - Get a random fact from any category

To get facts from a specific category, replace random with the desired category:

curl -H "Authorization: YOUR_API_KEY" "http://localhost:8080/api/fact?type=animals"

For example, to get a random animal fact, use type=animals.
